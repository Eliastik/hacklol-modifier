Documentation de la fonction « User Script » d'Hacklol Modifier

Voici la documentation de la fonctionnalité « User script » d'Hacklol. Cette fonctionnalité vous permet de créer votre propre script utilisateur, en utilisant le langage de programmation Javascript et les ressources disponibles par défaut dans Hacklol Modifier. Vous pouvez créer de nouveaux outils, améliorer les fonctionnalités par défaut, etc.

Cette fonction dispose d'un éditeur de colorisation syntaxique, CodeMirror, pour faciliter l'édition de scripts.

Une grande partie d'Hacklol Modifier est programmée en JavaScript. Toutes les fonctions essentielles sont intégrés dans un objet hacklol, afin de faciliter le développement de nouvelles fonctionnalités.

Hacklol Modifier dispose également de la bibliothèque jQuery (documentation ici : https://api.jquery.com/), vous pouvez donc utiliser toutes les fonctions disponibles avec cette bibliothèque. Hacklol Modifier dispose aussi de Magnific-Popup (documentation ici : http://dimsemenov.com/plugins/magnific-popup/documentation.html) pour l'affichage de popup et de jStorage (documentation ici : http://www.jstorage.info/) pour le stockage de données local (localStorage), ainsi que d'autres scripts (voir la fenêtre « A propos »).

Les scripts utilisateurs sont exécutés depuis une fonction anonyme. Cela signifie que toutes les variables créées sont supprimées une fois le script exécuté. Vous pouvez cependant modifier les fonctions et variables de l'objet hacklol, voire créer de nouvelles propriétés pour cet objet, puisque cela ne sera pas supprimé à la fin de l'exécution.

Ci-dessous, la liste des fonctions utilisables, commentées.

Fonctions, codes

Description, utilité

Paramètres, valeurs de retour

hacklol

Objet contenant toutes les fonctions d'Hacklol Modifier.


hacklol.description

Variable contenant une description d'Hacklol Modifier. Non utilisé par défaut.

Retourne une chaîne de caractère correspondant à une description d'Hacklol.

hacklol.version

Variable contenant la version d'Hacklol Modifier.

Retourne une chaîne de caractère correspondant à la version d'Hacklol Modifier exécutée.

hacklol.dateVersion

Variable contenant la date de sortie de la version actuelle d'Hacklol Modifier.

Retourne une chaîne de caractère correspondant à la date de sortie de la version actuelle d'Hacklol Modifier, sous la forme J/M/A.

hacklol.lang

Variable correspondant à la langue d'Hacklol Modifier. Utilisé uniquement pour les messages renvoyés par les scripts JavaScript.

Retourne une chaîne de caractère contenant le sigle de la langue. Valeurs possibles : fr, en.

hacklol.asciiArt

Variable contenant le titre « Hacklol » en version « Ascii Art ». Utilisé de manière discrète dans un endroit bien caché.

Retourne une chaîne de caractère contenant l'Ascii Art formaté pour s'afficher correctement (avec des entrées \n).

hacklol.tools

Variable contenant le nombre d'outils d'Hacklol Modifier. Non utilisé par défaut.

Retourne un nombre.

hacklol.urlPage

Variable contenant l'adresse URL du site choisi avant l'accès à Hacklol Modifier.

Retourne une chaîne de caractère.

hacklol.console()

Affiche un message dans la console JavaScript.

Aucun paramètre nécessaire. Ne retourne rien.

hacklol.rgb2hex(rgb)

Convertit une couleur RGB en codage hexadécimal.

@rgb : chaîne de caractère contenant une couleur au format RGB.

Retourne une chaîne de caractère contenant la couleur au format hexadécimal (sans le caractère # au début).

hacklol.getContrastYIQ(hexcolor)

Permet de savoir quelle couleur de texte (entre blanc ou noir) placer sur un fond de la couleur définit par le paramètre.

@hexcolor : chaîne de caractère contenant une couleur au format hexadécimal.

Retourne une chaîne de caractère : black ou white.

hacklol.checkVersion()

Envoi une requête Ajax au serveur d'Hacklol pour vérifier si la version d'Hacklol Modifier actuelle correspond bien à la dernière version. Dans le cas contraire, une texte est affiché dans la fenêtre « A propos » indiquant à l'utilisateur la marche à suivre pour obtenir la dernière version d'Hacklol Modifier.

Aucun paramètre nécessaire. Ne retourne rien.

hacklol.probabilite(min, max)

Tire au sort un nombre entier entre min et max.

@min : nombre entier correspondant au nombre minimal à tirer
@max : nombre entier correspondat au nombre maximal à tirer.

Retourne un nombre entier aléatoire entre min et max (inclus).

hacklol.loadPage(url, withHacklolPL)

Charge une url dans l'iframe principale d'Hacklol Modifier.

@url : chaîne de caractère correspondant à l'adresse URL du site à charger

@withHacklolPL : booléan indiquant si oui ou non il faut charger cette URL avec Hacklol Page Loader (côté serveur).

Ne retourne rien.

hacklol.reloadPage()

Recharge le site en arrière-plan. Cette fonction stoppe également l'outil « Modifier la page » pour éviter des bugs.

Retourne true.

hacklol.creerOutil(titre, fonction, fermerFenetre, idButton, style_css)

Ajoute un bouton dans la fenêtre « Boite à outils » et définit ses propriétés.



Attention : assurez-vous que cette fonction ne soit exécutée qu'une seule fois pour chaque outil que vous voulez créer, car elle ajoute un bouton à chaque appel sans faire attention à ce que l'outil existe déjà ou pas.

@titre : chaîne de caractère – titre de l'outil

@fonction : chaîne de caractères correspondant à une ou plusieurs instructions qui seront exécutées au clic sur le bouton (onclick)

@fermerFenetre : booléen – true si la fenêtre doit se fermer au clic sur le bouton, sinon false

@idButton : chaîne de caractère – id HTML du bouton

@style_css : chaîne de caractère définissant le style CSS supplémentaire du bouton – à noter que le bouton créé est l’élément <a> en HTML (type inline)

Tous les paramètres à part titre sont optionnels. Retourne toujours true.

hacklol.blinkFontBSOD()

Fait clignoter un faux curseur dans l'outil « BSOD ». Appelle l'autre fonction hacklol.setblinkFontBSOD() via un setTimeout() ;

Ne retourne rien.

hacklol.setblinkFontBSOD()

Fait clignoter un faux curseur dans l'outil « BSOD ». Appelle l'autre fonction hacklol.blinkFontBSOD() via un setTimeout() ;

Ne retourne rien.

hacklol.shake(time)

Fait trembler toute l'interface d'Hacklol Modifier. Utilisé dans l'outil « Faire exploser la page » pour une durée de 3 secondes (3000 millisecondes).

@time : nombre de millisecondes durant lesquelles l'effet doit se produire. Si le paramètre est omis, l’interface tremblera indéfiniment.

Retourne true.

hacklol.loadImages()

Charge les images d’Hacklol Modifier. Utilisé lors du lancement de l’application.

Retourne true.

hacklol.completeLoading()

Fonction exécutée lors de la fin du chargement des images d’Hacklol Modifier.

Ne retourne rien.

Fonctions bloquées

L'exécution des fonctions suivantes est bloqué :

  • eval() et new function ;

  • les fonctions de requêtes Ajax : tout ce qui contient xhr, ajax, $.post, $.get, XMLHttpRequest et XMLHTTP est bloqué ;

  • L'objet document.cookie est bloqué (vous pouvez toujours utiliser le localStorage) ;

  • Les fonctions et variables suivantes sont bloquées : userScriptFunction, codeMirrorUserScript et execUserScript.

Si ces fonctions sont détectées, l'exécution de votre script sera annulée (mais le script sera quand même enregistré).
